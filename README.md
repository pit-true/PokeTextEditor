# 点字
何かしら点字を読んでもらえると分かりますが、RSEでは最初の6byteでメッセージウィンドウの表示位置（表示サイズ）、点字の座標を設定しています。
例えばこじまのよこあなの点字(EM:0x26377a)だと、最初の6byteは 05 00 19 13 08 03 となっています。

[枠左辺位置][枠上辺位置][枠右辺位置][枠下辺位置][文字x座標][文字y座標]の並びになっているので、RSEで使う際は必ず6byte分入れてください。
PTEでは<00>のように囲むことで、16進数変換しても（書き込んでも）00とそのままバイナリに変換します。

[枠左辺位置]、[枠右辺位置]は数値を小さくするとメッセージウィンドウが左上に広がり、[枠上辺位置][枠下辺位置]は数値を小さくするとメッセージウィンドウが狭まります。

[枠右辺位置] - [枠左辺位置] > 0 かつ [枠下辺位置] - [枠上辺位置] > 0 を満たせるようにしてください。
そうしないとメッセージウィンドウのサイズが負の値を取るためバグります。
先ほど挙げた 05 00 19 13 08 03 を例にとると、メッセージウィンドウのサイズは0x08(0x13-0x05)*0x13(0x13-0x00)を取り、x座標0x08, y座標0x03から点字を表示します。

# 入力上の注意

漢字パッチを当てていると記述するときに以下の表現が被ります。

--------+-----------
   b7=円|fc 89 7e=円
   f1=人|fc 90 6c=人
   f2=形|fc 8c 60=形
   f5=勝|fc 8f 9f=勝
   f6=負|fc 95 89=負
f9 00=↑|fc 81 aa=↑
f9 01=↓|fc 81 ab=↓
f9 02=←|fc 81 a9=←
f9 03=→|fc 81 a8=→
f9 13=（|fc 81 69=（
f9 14=）|fc 81 6a=）
f9 15=◎|fc 81 9d=◎
f9 16=△|fc 81 a2=△
--------+-----------

f9 XXで始まる記号を使いたい場合はそのまま入力してください。
moji.xmlでは3byteの方を{円}のように登録しています。
漢字パッチの方の「円」やfc（漢字パッチ）で始まる記号を入力したい場合は入力時に{}で囲ってください。

入力の例)	円{円}↑{↑}→{→}
出力結果) 	b7 fc 89 7e f9 00 fc 81 aa f9 03 fc 81 a8

被っている記号を{↑}等で登録しているだけなので、変更したければ各自で変更を加えて頂ければ幸いです。
基本的にどちらもFR上での見た目は変わりませんので、普通に使う場合は{}を付けずにそのまま入力すればいいと思います。

被っていない以下の記号は{}で囲わずそのまま入力してください。
＋, ①, ②, ③, ④, ⑤, ⑥, ⑦, ⑧, ⑨ , ＿

また、ステータス画面等で使われている以下の記号を使用したい場合も{}で囲ってください。

---------
F9 05=Lv
F9 06=PP
F9 07=ID.
F9 08=No
---------

入力の例)	Lv56{Lv}56
出力結果)	c6 ea a6 a7 f9 05 a6 a7

文字色変更も今までは色白、色黒という表記でしたが今回は{白}{黒}のような表記になりました。
詳しくはmoji.xmlを覗いてみてください。
fc 01:{文字色}
fc 02:{背景色}
fc 03:{影色}
fc 04:{一括変更}
fc 06:{フォント}
fc 08:{ウェイト}
fc 09:{AB待ち一時停止}
fc 0a:{音待ち一時停止}
fc 0b:{BGM変更}
fc 0c:{記号表示}
fc 0d:{x座標}
fc 0e:{y座標}
fc 0f:{文字消去}
fc 10:{BGM再生}
fc 17:{BGM一時停止}
fc 18:{BGM再開}

Q.ウェイトとか後ろは変数だけど、どう入力するの？
A.{ウェイト}<01>のように入力してください。
